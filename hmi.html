Live gauges
All sensor channels displayed in real time. Each gauge shows a live bar, value, unit, and configurable range. Colour-coded by sensor type. Additional channels added per bench configuration.
Performance curve
Flow vs pressure plotted live as the test runs. Specification limit bands shown in green. Cursor tracks the current reading. Torque reference overlay available. Zoom and pan after test completes.
Camera feed
Live view of the pump under test. Recording starts and stops with the test session. Video is time-synchronised with sensor data and embedded in the final PDF report.
E-stop
Emergency stop halts all systems immediately — closes valves, kills the pump motor, and saves all recorded data to that point. Clearly labelled with the exact consequence so any operator understands it instantly.
Test details
Customer name, job number, component description, part number, and operator all captured before the test begins. These appear on the top bar during the test and are printed directly onto the PDF certificate.
Alarm system
Multi-level alarm monitoring runs as a background process throughout the test. Warnings appear in the gauge panel. Critical alarms trigger the E-stop automatically. All events are logged with a sensor snapshot.
